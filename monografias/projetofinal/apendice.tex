\section{Descrição textual dos casos de uso}

\subsection{Segmentação da Imagem}

\subsubsection*{Sumário}

Este caso de uso se inicia quando o operador aciona o software, inserindo como entrada o endereço de um arquivo de imagem ou um grupo de imagens. O objetivo deste caso de uso é subdividir uma imagem e, de acordo com o algoritmo escolhido para teste, aplicá-lo a cada quadro dessa divisão e gerar um resultado final, após uma segmentação completa da imagem.

\subsubsection*{Pré-condições}

\begin{itemize}
	\item Os arquivos de imagens inseridos devem existir no endereço definido e devem estar em um dos formatos conhecidos e suportados pelas bibliotecas utilizadas pelo sistema.
	\item Os tamanhos dos quadros em pixels, fatores e configurações necessárias para execução dos algoritmos devem estar definidas.
\end{itemize}

\subsubsection*{Fluxos}

Este caso de uso se inicia quando o operador aciona o software e inserindo como entrada o endereço de um arquivo de imagem ou um grupo de imagens.
\begin{enumerate}
	\item O operador insere no software uma imagem ou grupo de imagens a ser(em) segmentada(s).
	\item O operador define o algoritmo a ser aplicado na imagem para cálculo do limiar a ser aplicado na segmentação.
	\item O software abre a imagem e a converte para Gray Scale (Escala de Cinza).
	\item O software divide a imagem em quadros, de acordo com a altura e largura em pixels definidas para cada quadro; respeitando a condição dos quadros da última coluna e última linha que, em determinados casos, terão tamanho diferenciado.
	\item O software percorre cada quadro da divisão da imagem, definindo a região a ser aplicada.
	\begin{enumerate}
		\item Para cada quadro da imagem, o software calcula o limiar de acordo com o algoritmo definido. Neste ponto, o software aciona o caso de uso referente ao algoritmo escolhido.
		\item software aplica na região do quadro o algoritmo de segmentação (Threshold), usando o limiar encontrado.
		\item O software soma a um total geral da imagem os valores encontrados para foreground.
	\end{enumerate}
	\item O software exibe um relatório com os valores absolutos e relativos encontrados na segmentação, como valor do total de foreground e background calculados.
\end{enumerate}

\subsubsection*{Pós-condições:}

\begin{itemize}
\item Exibir na tela uma foto da imagem, com os contornos encontrados na segmentação da mesma.
\end{itemize}

\subsection{Cálculo de Limiar por Média}

\subsubsection*{Sumário}
Este caso de uso acontece para cada quadro da imagem, sendo acionado no passo 5.1 do caso de uso 1 - Segmentação da imagem, se o algoritmo escolhido for Média. O objetivo deste caso de uso é definir um limiar para ser aplicado na segmentação, com base no cálculo da média  entre os pixels da região.

\subsubsection*{Pré-condições}

\begin{itemize}
	\item A imagem deve estar dividida em quadros e a região de imagem a ser aplicado algoritmo já deve estar definida.
	\item O caso de uso  1 - Segmentação da imagem deve estar sendo executado no 5º passo
\end{itemize}

\subsubsection*{Fluxos}

Este caso de uso acontece para cada quadro da imagem, sendo acionado no passo 5.1 do caso de uso 1 - Segmentação da imagem, se o algoritmo escolhido for Média.

\subsubsection*{Curso normal}

O software calcula a média dos valores dos pixels do quadro.

\subsubsection*{Pós-condições:}

\begin{itemize}
\item Retornar o limiar encontrado com base na média de todos os pixels do quadro.
\end{itemize}

\subsection{Cálculo de Limiar Adaptativo:}

\subsubsection*{Sumário}

Este caso de uso acontece para cada quadro da imagem, sendo acionado no passo 5.1 do caso de uso 1 - Segmentação da imagem, se o algoritmo escolhido for Limiar Adaptativo. O objetivo deste caso de uso é definir um limiar para ser aplicado na segmentação, com base no cálculo de um Limiar Adaptativo sobre os pixels da região.

\subsubsection*{Pré-condições}
	\begin{itemize}
		\item A imagem deve estar dividida em quadros e a região de imagem a ser aplicado algoritmo já deve estar definida.
		\item O caso de uso  1 - Segmentação da imagem deve estar sendo executado no 5º passo
	\end{itemize}

\subsubsection*{Fluxos}

Este caso de uso acontece para cada quadro da imagem, sendo acionado no passo 5.1 do caso de uso 1 - Segmentação da imagem, se o algoritmo escolhido for Limiar Adaptativo. 

\subsubsection*{Curso normal}

\begin{enumerate}
	\item O software calcula a média dos valores dos pixels do quadro
	\item O software calcula a média dos valores que estão abaixo da média encontrada e a média dos valores que estão a partir da média encontrada.
	\item O software calcula a média dos dois valores encontradose compara com o a média anterior
	\item O software repete dos passos 2 ao 3 enquanto a diferença entre a média atual e a média anterior for maior do que um valor predefinido nas configurações do sistema.
	\item Ao sair do loop anterior, o software retorna o resultado encontrado como limiar para cálculo da segmentação do quadro atualmente definido.
\end{enumerate}

\subsubsection*{Pós-condições:}

\begin{itemize}
	\item Retornar o limiar encontrado com base na média de todos os pixels do quadro.
\end{itemize}

\subsection{Cálculo de Limiar por Entropia de Shannon}

\subsubsection*{Sumário}
Este caso de uso acontece para cada quadro da imagem, sendo acionado no passo 5.1 do caso de uso 1 - Segmentação da imagem, se o algoritmo escolhido for segmentação por Entropia de Shannon. O objetivo deste caso de uso é definir um limiar para ser aplicado na segmentação, com base no cálculo utilizando segmentação por Entropia de Shannon nos pixels da região.

\subsubsection*{Pré-condições:}

\begin{itemize}
	\item A imagem deve estar dividida em quadros e a região de imagem a ser aplicado algoritmo já deve estar definida.
	\item O caso de uso  1 - Segmentação da imagem deve estar sendo executado no 5º passo
\end{itemize}

\subsubsection*{Fluxos}

Este caso de uso acontece para cada quadro da imagem, sendo acionado no passo 5.1 do caso de uso 1 - Segmentação da imagem, se o algoritmo escolhido for Segmentação por Entropia e Shannon. 

\subsubsection*{Curso normal}

O software calcula o limiar, baseado na segmentação por Entropia de Shannon sobre os valores dos pixels do quadro selecionado.

\subsubsection*{Pós-condições:}

Retornar o limiar encontrado com base na segmentação por Entropia de Shannon sobre os pixels do quadro selecionado.
